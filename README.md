**Моя первая игра на pygame** (по вселенной stalker). 


Главный герой попадает в пространственную аномалию и ему нужно оттуда выбратся, но если он зайдет в туман его вернет в ту же точку откуда он начал. 

Чтобы победить нужно найти артефакт - компас, по возможности убить всех врагов. Можно так же подобрать бонус, который поможет в прохождении. 

**УПРАВЛЕНИЕ:**
  
  Стрелка вверх - идти вверх
  
  Стрелка вниз - идти вниз
  
  Стрелка влево - идти влево
  
  Стрелка вправо - идти вправо
  
  **СТРЕЛЬБА:**
  
  W - Выстрел вверх
  
  D - Выстрел вправо
  
  S - Выстрел вниз
  
  A - Выстрел влево
